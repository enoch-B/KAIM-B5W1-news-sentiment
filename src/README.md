# Quantitative Analysis with TA-Lib and PyNance

This project uses technical indicators to analyze stock data using TA-Lib and PyNance.

## Setup
```bash
pip install -r requirements.txt
```

## Run
```bash
python src/main.py
```

## Structure
- `data/` contains raw and processed data
- `src/` contains modular code for loading, analyzing, and visualizing
- `outputs/` for charts and reports

## KPIs
- Applied technical indicators accurately
- Complete EDA pipeline
- Modular structure for reproducibility
